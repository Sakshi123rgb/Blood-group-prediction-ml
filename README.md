# 🩸 Blood Group Prediction using Machine Learning

✨ *An end-to-end Machine Learning project to predict blood groups using synthetic medical data.*

---

## 🚀 Project Highlights
✔️ Complete ML Pipeline  
✔️ Data Generation using NumPy  
✔️ Data Preprocessing & Encoding  
✔️ Model Training using Random Forest  
✔️ Accuracy Evaluation  
✔️ Prediction on New Data  
✔️ Data Visualization  

---

## 📊 Dataset Information

🔹 The dataset is **synthetically generated** using NumPy  
🔹 It contains medical-related features:

| Feature      | Description |
|-------------|------------|
| Hemoglobin  | Hemoglobin level in blood |
| RBC         | Red Blood Cells count |
| WBC         | White Blood Cells count |
| Platelets   | Platelet count |
| Age         | Age of the patient |

🎯 **Target Variable:** Blood Group (A, B, AB, O)
